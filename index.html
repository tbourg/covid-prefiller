Comment utiliser ce bouton ?

Sur ordinateur
  1. Déplacez ce bouton dans votre barre de favoris.
  2. Rendez-vous sur la page du gouvernement
  3. Cliquez sur le favori créé.
Sur mobile
(Un peu plus compliqué !)
  1. Appui long sur le bouton et « copier le lien de l’url».
  2. Ajoutez cette page en favoris
  3. Modifiez le favori et remplacez l’url du favori par le lien copié.
  4. Puis rendez-vous sur la page du gouvernement
  5. Ecrivez les premières lettres du nom du favori () dans la barre d'adresse
  6. Cliquez sur le favori dans les suggestions de la complétion automatique.
